# Licensing Plan

Use unmodified GNU AGPL-3.0.

Rationale:

- Network-facing server software
- Modified public deployments must offer corresponding source
- Commercial use remains allowed, preserving open-source status
- Closed-source hosted forks are discouraged

Use a small, reasonable Section 7-compatible attribution notice, subject to legal review:

> Based on FreeTier Atlas, originally created by Sergiy. Source and licence information are available from the application's About/Legal page.

Repository files:

- `LICENSE`
- `NOTICE`
- `ADDITIONAL_TERMS.md`
- `TRADEMARKS.md`
- `AUTHORS.md`
- `THIRD_PARTY_NOTICES.md`
- `CONTRIBUTING.md`

Preserve MIT notices for substantial reused material. Do not copy from repositories without an explicit licence.
