# Roadmap

## M0 Foundation

Repository, licence, monorepo, CI, Compose skeleton, config loader, ADRs.

## M1 Cloudflare vertical slice

Data model, adapters, evidence, verification, API, provider page, admin review, RSS.

Exit: one provider works end to end.

## M2 Catalogue UX

Homepage, search/filter/sort, comparison, evidence, history, completeness.

## M3 Adviser

Requirements, deterministic matching, quotas, reductions, self-host fallback, LLM routing, ZIP.

## M4 Seven providers

GitHub, AWS, Google, Azure, Vercel, Oracle.

## M5 Operations and hardening

Admin, Discord, abuse controls, privacy, retention, observability, accessibility.

## M6 Public Z0 proof

Cloudflare Pages primary, GitHub Pages mirror, dynamic Z0 components, quota dashboard, reproducible deployment.

## Post-MVP

Web push, Terraform/OpenTofu, Ansible, generated GitHub Actions, Kubernetes/Helm, 20–30 providers, filtered RSS, provider SDK.
