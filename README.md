# FreeTier Atlas

FreeTier Atlas is an open-source, evidence-backed catalogue and architecture adviser for cloud and developer services that can be used at zero cost.

It distinguishes:

- **Z0 — True $0:** no credit card, no required purchase, no automatic billing, and a hard stop, throttle, sleep, or rejection when quotas are exhausted.
- **Z1 — Free allowance with billing exposure**
- **Z2 — Temporarily or conditionally free**
- **Z3 — Free open-source software requiring infrastructure**

## MVP providers

Azure, AWS, Google Cloud, Oracle Cloud, Cloudflare, Vercel, and GitHub.
